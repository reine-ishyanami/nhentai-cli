# Version 0.1.0 (2024-08-12)

1. generate profile

   ```bash
   nhentai-rs generate
   ```

2. download 

   ```bash
   nhentai-rs download --name <name>
   ```

3. convert to pdf

   ```bash
   nhentai-rs convert --path <path> --name <name> [--dir <dir>]
   ```

4. compress

   ```bash
   nhentai-rs compress --path <path> --name <name> [--secret <secret> --dir <dir>]
   ```
